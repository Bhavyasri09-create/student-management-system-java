# Student Management System (Java)

A **menu-driven console app** in Java to manage students.  
Supports **add, view, search, delete** operations with **alphanumeric IDs**.

---

## Features
- Add, view, search, delete students  
- Alphanumeric Student IDs  
- Input validation to prevent errors  

---

## How to Run
```bash
javac StudentManagementSystem.java
java StudentManagementSystem
